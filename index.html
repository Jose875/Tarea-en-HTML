One of the most powerful features of style sheet programming is to define the styles of a whole website. This is achieved by creating a file where we only place the style declarations of the page and linking all the pages of the site with that file. In this way, all the pages share the same style declaration and, therefore, if we change it, all the pages will change.

Cascading Style Sheets are written within the HTML code of the web page, only in advanced cases can be written in a separate file and link the page with that file. In the beginning we will use the most direct way to apply styles to the elements of the page,

The benefit of using responsive design on a website is mainly that the website automatically adapts to the device from which it is accessed. PHP is the acronym for Hipertext Preprocessor. It is a free and platform-independent server-side programming language, fast, with a large library of functions and a lot of documentation.
A server-side language is one that runs on the web server, just before the page is sent over the Internet to the client. The pages that run on the server can perform access to databases, network connections, and other tasks to create the final page that the client will see. The client only receives a page with the HTML code resulting from the execution of the PHP. Since the resulting page contains only HTML code, it is compatible with all browsers.

PHP, in the case of being mounted on a Linux or Unix server, is faster than ASP, since it runs in a single memory space and this prevents communications between COM components that are made between all the technologies involved in a page ASP.

Finally we pointed out the security, in this point it is also important the fact that in many cases PHP is installed on Unix or Linux servers, which are better known as faster and safer than the operating system where ASP runs, Windows NT or 2000. In addition, PHP allows you to configure the server to allow or reject different uses, which can make the language more or less secure depending on the needs of each.
